# To-Do List GUI

A simple To-Do List desktop application built with Python and Tkinter that allows users to manage tasks with automatic file persistence.

This project was built from scratch as part of my Python learning journey. It allowed me to practice file handling, GUI development with Tkinter, Git/GitHub, and fundamental software development concepts.

## Project Status

🚧 In development

## Technologies

- Python
- Tkinter
- Git
- GitHub

## Current Features

- ✅ Load tasks from a text file
- ✅ Add new tasks
- ✅ Delete selected tasks
- ✅ Automatically save tasks
- ✅ Input validation

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ainhoazagarzazu/todo_gui_python.git
```

2. Navigate to the project folder

```bash
cd todo_gui_python
```

3. Run the application

```bash
python todo_gui.py
```
